# Portfolio Construction and Optimization

Course Project — King’s College London  
**Period:** February 2025 – May 2025  

---

## Overview

This project implements a **Python-based portfolio optimization framework** designed to analyze and optimize Exchange Traded Funds (ETFs), focusing on **SPY** and **QQQ**.  
The pipeline integrates time series resampling, performance metrics, and optimization techniques to construct efficient portfolios under various objectives.

---

## Objectives

- Compute **performance and risk metrics**: drawdown, Sharpe ratio, Sortino ratio, and rolling volatility.  
- Compare ETF returns with benchmarks (S&P 500, HFRI CTA).  
- Apply **OLS regression** for performance attribution.  
- Implement **Markowitz mean-variance optimization** for:  
  - Minimum variance portfolios  
  - Maximum return portfolios  
  - Maximum Sharpe ratio portfolios  
- Plot the **efficient frontier** and **portfolio weight heatmaps** for visualization.
